# Library Management System

A simple **Python-based Library Management System** demonstrating OOP principles, repository pattern, and in-memory storage.

## Features
- Add / list books and members
- Borrow and return books
- Track active loans
- CLI interface for easy interaction
- Exception handling for invalid operations

## Tech Stack
- Python 3.13
- Dataclasses
- Repository & Service design patterns

## Usage
Run the CLI:

```bash
python cli.py
